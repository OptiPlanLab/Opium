# OPIUM — OptiPlan Business Advisor Guide

> **Interactive capabilities guide for OptiPlan's AI business advisor agent.**

## What is OPIUM?

OPIUM (**O**pti**P**lan **I**ntelligence **U**nit for **M**anagement) is OptiPlan's dedicated AI business advisor — a strategic partner built on Claude (Anthropic) that knows OptiPlan's product, market, competitors, fundraising goals, and founder decision patterns inside out.

OPIUM operates as a Claude Project with 12 knowledge files covering:
- Product encyclopedia (8 modules across 3 pillars)
- Market & competitive intelligence (8+ competitors profiled)
- Pricing & business model scenarios
- Investor deck gap analysis & pitch narrative
- Founder communication patterns
- Technical architecture (business-framed)
- User personas, success metrics & KPIs
- Hebrew terminology reference
- Design specification for branded HTML outputs

## This Repository

This repo contains the **OPIUM Capabilities Guide** — a single-page interactive HTML document designed for OptiPlan's co-founders (Amit Zomerfreund & Maor Dahan) to understand how to work with OPIUM effectively.

### What the Guide Covers

1. **Who OPIUM is** — cognitive style, core principles, how it thinks
2. **What OPIUM knows** — 12 knowledge files, 8 products, 8+ competitors, 6 personas
3. **Capabilities** — Strategy & advisory, content production, research & analysis, investor prep
4. **Quick start prompts** — Click-to-copy prompts for immediate use
5. **Document types** — All deliverables OPIUM can produce (HTML pages, financial models, slide drafts, handoff docs)
6. **Founder profiles** — How OPIUM adapts communication to Amit vs. Maor
7. **Company knowledge** — Products, market, fundraising, deck gaps, competitors
8. **Boundaries** — What OPIUM does and does not handle

## How to Use

### View the Guide
```bash
# Clone the repo
git clone https://github.com/OptiPlanLab/opium-guide.git

# Open in browser
open opium-guide.html
```

Or simply download `opium-guide.html` and open it in any modern browser. No server required — it's a self-contained HTML file.

### Use OPIUM
1. Go to [claude.ai](https://claude.ai)
2. Open the **OPIUM** project
3. Start a conversation with a specific question or task
4. Copy any prompt from the guide's "Quick Start" section to get going

## Tech Stack

The guide is a single HTML file using:
- **Google Fonts** — Playfair Display + Source Sans 3
- **Vanilla CSS** — OptiPlan design system (navy/steel/sky/ice palette)
- **Vanilla JS** — Tabs, accordion, click-to-copy
- **No dependencies** — No frameworks, no build step, no npm

## Design

Follows the OptiPlan brand specification:
- **Palette:** Navy `#1B365D` · Steel `#4A86B8` · Sky `#89C4E8` · Ice `#D6EAF5`
- **Typography:** Playfair Display (headings) · Source Sans 3 (body)
- **Aesthetic:** "Polo Ralph Lauren" — understated, confident, premium. Clean typographic hierarchy, generous whitespace, subtle borders over heavy shadows.
